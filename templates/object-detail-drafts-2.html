Object Detail Layouts — Part 2

Four more layout suggestions for displaying grouped object properties.

5 Split Card — Left Header / Right Properties

Each group is a card split into two columns. The left side contains the group title, description, and an icon. The right side lists all properties as label-value rows.

6 Accordion — Collapsible Groups with Summary Badges

Each group is an accordion item that can be expanded or collapsed. A badge in the header shows the property count. Annotations appear as muted text below values.

8 Striped Rows — Compact Grouped Sections with Alternating Rows

Groups are separated by a bold header bar. Properties are displayed in alternating striped rows for readability. Annotations are shown inline as italic text.
